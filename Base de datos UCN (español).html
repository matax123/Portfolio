Base de datos UCN

Base de datos UCN es una aplicación de escritorio trabajada en C#, la cual posee una interfaz gráfica hecha con windows forms.

El programa fue un trabajo que nos pidieron a los estudiantes de ingeniería en informática de la UCN para el ramo "bases de datos". El trabajo incluía un informe con la documentación interna y guía del usuario.
